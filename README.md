# 📚 The Reader

A modern Android reading application built with Kotlin that provides a smooth and clean book reading experience.  
The app focuses on performance, usability, and simple UI interactions for an enjoyable reading workflow.

---

## ✨ Features

- 📖 Browse and read books
- 🔍 Search functionality
- ❤️ Save favorite books
- 📚 Book details screen
- ⚡ Smooth and responsive UI
- 🎨 Modern Material Design
- 📱 Optimized Android experience

---

## 🛠️ Technologies Used

- Kotlin
- Android SDK
- Jetpack Compose
- MVVM Architecture
- Coroutines
- Firebase
- REST API Integration

---

## 🧠 Concepts Demonstrated

- State management
- API integration
- Modern Android architecture
- Asynchronous programming with Coroutines
- Clean and maintainable code structure
- Responsive UI design

---

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ithassan/the_reader_main.git
```

### Open in Android Studio

1. Open Android Studio
2. Select Open Project
3. Sync Gradle files
4. Run the application
